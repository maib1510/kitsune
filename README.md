# WatchAtlas

WatchAtlas is a web platform that helps users discover where anime and TV series are available across different streaming services.

Users can search for content, check availability on platforms such as Netflix, Crunchyroll, Prime Video or Disney+, and access the selected service directly. The application also includes personal watchlists, favorites, user profiles, social features and recommendation tools.

## Features

* Search anime and TV series
* View streaming platform availability
* Direct links to streaming services
* User authentication and profiles
* Favorites management
* Watchlists (Plan to Watch, Watching, Completed)
* Friend system
* Personalized recommendations
* Responsive and modern user interface

## Tech Stack

### Frontend

* React
* TypeScript
* Tailwind CSS

### Backend

* Spring Boot
* Spring Security
* JWT Authentication

### Database

* PostgreSQL

### External APIs

* AniList API
* TMDB API

## Project Goals

The goal of this project is to build a full-stack web application that combines content discovery, streaming availability tracking and social interaction in a single platform.
